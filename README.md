# 📊 Financial Analysis Dashboard – Sorority Expenses

## 📌 Introduction
A financial dataset from a shared living environment (sorority) was analyzed using Power BI Desktop to generate reports and uncover meaningful insights about spending behavior.

This project was developed as a practical application of data analysis concepts, focusing on transforming raw financial data into useful information.

---

## ❓ Problem Statement
The data was analyzed to provide answers to the following questions:

- How are expenses distributed across different categories?
- How do expenses change over time?
- How do spending patterns differ between residents?
- Are there any months with unusually high expenses?
- What is the average monthly spending?

---

## ⚙️ Methodology
The dataset was processed, analyzed, and visualized using:

- Python for data cleaning and consolidation;
- Power Query for data transformation and validation;
- Power BI for data modeling and visualization.

---

## 🛠️ Technologies Used

- Power BI  
- Power Query  
- DAX  
- Python (Pandas)  
- Microsoft Excel

## 📈 Summary and Conclusion
The analysis provided important insights into the financial behavior of the household.

Rent represents the largest portion of expenses, indicating a strong presence of fixed costs. Variable expenses show fluctuations across different months, suggesting opportunities for better control.

Additionally, differences in spending patterns between residents were identified, as well as specific months with higher levels of expenditure.

Overall, the project demonstrates how data analysis can support better financial understanding and decision-making.
